# CLENDARIO DE JORNADAS Y TABLA DE POSICIONES DE LA FUTVE 2

⚽ Calendario y Posiciones - Liga FUTVE 2 (Temporada 2026)
Este proyecto es una aplicación web interactiva diseñada para seguir de cerca la Temporada 2026 de la Liga FUTVE 2 (Segunda División de Venezuela). Permite consultar el calendario completo de las 18 jornadas y visualizar las tablas de posiciones actualizadas de los grupos Centro-Occidental y Centro-Oriental.

✨ Características Principaless
Calendario Completo: Visualización detallada de las 18 jornadas de ambos grupos.

Tablas de Posiciones Dinámicas: Cálculo automático de puntos, diferencia de goles (GD) y average basándose en los resultados cargados.

Navegación Intuitiva: Sistema de flechas y botones optimizado para móviles y computadoras.

Buscador en Tiempo Real: Filtra partidos por nombre de equipo (ej. Lara, Yaracuyanos, Aragua).

Exportación de Datos: Descarga el calendario y las tablas en formato Excel (.xlsx) o PDF profesional.

Diseño Moderno: Interfaz limpia construida con Tailwind CSS y Lucide Icons.

Protección de Interfaz: Mecanismos básicos para evitar la inspección accidental del código.

🚀 Tecnologías Utilizadas
HTML5 / CSS3

JavaScript (ES6+)

Tailwind CSS (Framework de estilos)

Lucide React (Iconografía)

SheetJS (Librería para exportación a Excel)

jsPDF & AutoTable (Generación de reportes PDF)

📦 Instalación y Uso
No requiere instalación de servidores pesados. Al ser un sitio estático, puedes ejecutarlo simplemente abriendo el archivo index.html en cualquier navegador o desplegarlo en GitHub Pages.

Clona el repositorio:

Bash
git clone https://ditt0tech.github.io/calendario2026futve2/
Asegúrate de que el archivo del logo (image_07ca51.png) esté en la misma carpeta raíz.

Abre index.html en tu navegador.
